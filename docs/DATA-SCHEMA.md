# Схема выгрузки CRM → Sheets → Power BI

## Листы назначения

| Лист | Содержимое |
|------|------------|
| Комнаты | id, статус, группа цвета, метрики |
| Приёмки fact | клиентские приёмки по датам |
| Сделки | даты ДДУ/ДКП при необходимости |

Колонки и стратегия — в `docs/BI_EXPORT.md`.

Секреты: `examples/crm_secrets.json.example`, service account JSON вне git.
